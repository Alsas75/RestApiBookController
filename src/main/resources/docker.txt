FROM базовый образ (openjdk)
WORKDIR КАТАЛОГ ВНУТРИ КОНТЕЙНЕРА
COPY копирует файлы в контейнер
RUN выполняет команды во время сборки
EXPOSE отрывает порт
ENTRYPOINT основная команда, которая будет выполняться


Жизненный цикл
1. Пишем код в Intellij
2. Собираем jar с помощью maven
3. Пишем Dockerfile
4. Собираем образ (image) через docker build
5. Запускаем контейнер через команду docker run
6. Если изменили код --> пересобираем
